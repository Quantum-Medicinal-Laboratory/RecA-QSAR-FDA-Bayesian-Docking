# RecA Inhibitor Discovery

This repository contains a publication-ready computational workflow for identifying potential Mycobacterium tuberculosis RecA inhibitors using:

- ChEMBL bioactivity data curation
- PaDEL fingerprint generation
- Feature selection and benchmarking
- Random Forest QSAR modeling
- External validation
- Y-randomization
- Applicability domain analysis
- Bayesian fingerprint classification
- FDA-approved drug screening
- Molecular docking

## Main Results

- ROC-AUC: 0.832
- PR-AUC: 0.883
- Accuracy: 0.778
- MCC: 0.526
- 5-Fold CV ROC-AUC: 0.837
- Bootstrap 95% CI: 0.752–0.905
- Y-randomization p-value: 0.0099
